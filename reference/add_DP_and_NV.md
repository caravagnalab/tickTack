# add_DP_and_NV Function

This function allows you to simulate the NV and DP for the CN events.

## Usage

``` r
add_DP_and_NV(karyotype, simulated_mutations, coverage, purity)
```

## Arguments

- karyotype:

  karyotype

- simulated_mutations:

  mutations

- coverage:

  coverage

- purity:

  purity

## Value

get mutation subfunction
