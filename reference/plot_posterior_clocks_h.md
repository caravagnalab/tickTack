# Plot posterior clocks distributions obtained from the hierarchical model fit

Plot posterior clocks distributions obtained from the hierarchical model
fit

## Usage

``` r
plot_posterior_clocks_h(results, K)
```

## Arguments

- results:

  list of 4: \$data, \$draws_and_summary, \$log_lik_matrix_list and
  \$elbo_iterations

- K:

  index of inference whose results want to be plotted

## Value

areas_tau
