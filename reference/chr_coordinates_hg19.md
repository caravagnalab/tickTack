# hg19 coordinates

Reference coordinates (hg19)

## Usage

``` r
chr_coordinates_hg19
```

## Format

### `chr_coordinates_hg19`

A data frame:

- chr:

  chromosome identifier

- length:

  length of chromosome

- from:

  Absolute coordinate of chromosome start

- to:

  Absolute coordinate of chromosome end

- centromerStart:

  Absolute coordinate of chromosome centromere start

- centromerEnd:

  Absolute coordinate of chromosome centromere end
