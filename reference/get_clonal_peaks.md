# get_clonal_peaks Function

Obtain the theoretical peaks that would be observed in a VAF spectrum.

## Usage

``` r
get_clonal_peaks(k, purity)

get_clonal_peaks(k, purity)
```

## Arguments

- k:

  karyotype

- purity:

  peaks

## Value

get the expected VAF peaks from the karyotype and the sample purity
value
